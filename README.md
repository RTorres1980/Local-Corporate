# Retail & Manufacturing Financial Intelligence Suite

### **Overview**
This project simulates a corporate finance environment for a multi-unit retail and manufacturing entity. It automates the extraction, transformation, and visualization of ERP data to drive strategic decision-making.

### **Key Features**
* **Automated Financial Reporting:** Integrated P&L and Balance Sheet reconciliation using synthetic GL data.
* **Inventory Optimization:** Calculated **Inventory Turnover** and **ROI per SKU** to identify slow-moving manufacturing assets.
* **KPI Tracking:** Dynamic dashboards for Gross Margin analysis and Store-Level profitability.
* **Forecasting Model:** A 12-month rolling forecast built in Excel/Power BI using comparative performance metrics.

### **Tech Stack**
* **Data Visualization:** Power BI (DAX, Star Schema)
* **Data Engineering:** Python (Pandas, NumPy) for ETL
* **Financial Modeling:** Advanced Excel (Power Query)
* **Accounting Standards:** Built following GAAP principles for revenue recognition and expense accrual.
